Overview

After completing the assigned reading for the week and your team activity, you are prepared to complete your personal assignment. Your assignment will be done in the repo you downloaded as part of your team activity and will consist of creating a form, sending data to the server, then displaying that data in a different part of your webpage. This will help you learn how to work with git, data, pages, and client-server communication.

Instructions

Your first assignment will be to create a server that handles at least two different endpoints.

  1. Create prove01.js (name is up to you) file in the routes folder. (Folder already exists from download in team activity)

  2. Create a route to your prove01.js file in your index.js file

  3. In your views folder create a two .ejs files (Or pug/handlebars if you chose to use either of those)

    • One view with a simple form with two inputs

    • Another view to display the input back to the user

  4. Parse the users input on the server and send the input to second view

  5. Be sure to use nodemon

  6. Push your code up to Github.

    7. If you would like help getting started with git this video may be helpful.

  7. Please review rubric below to see how you'll be graded and to ensure you've completed each aspect of this assignment.
