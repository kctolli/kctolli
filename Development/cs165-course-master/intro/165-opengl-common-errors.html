OpenGL - Common Errors

Here are some errors and advice for resolution from past students:

  • USB error:

    If you run into an error about USB 2.0, use settings to enable USB 1.0 only instead

    Instructions: Click settings Icon in upper left hand corner > select USB on the left side > Select USB 1.1 then Hit OK

  • Can't connect to linux lab

    Instructions: If you run into an error connecting to the linux lab, use the following:

    Host: sftp://157.201.194.201 Port: 215 and use your linux username and password

    Then click QuickConnect . You are now connected. Remote files on the right, bottom pane at /home/cs165new/uiTest.

  • Can't find local directory or copy files to local directory

    Instructions: You need to create the local directory. You can do that either with a command line window, or right-clicking the cs165 directory in filezilla and selecting Create New Directory.
