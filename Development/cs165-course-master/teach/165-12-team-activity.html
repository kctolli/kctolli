12 Teach : Team Activity - Operator Overloading

Overview

This activity will help you practice the syntax of operator overloading.

Your assignment is to overload operators for a basic Point and Velocity class.

The velocity class you have been working on in your projects is likely more sophisticated than the simple one you will be provided with. The hope is that as a team, you can overload some operators for this class that may prove helpful in your projects. Then you can individually implement these into your own project code.

Instructions

Begin with the code at: /home/cs165new/ta12/*

This code includes the following two classes:

Point
-x : float
-y : float
+getX() : float
+getY() : float
+setX(float) : void
+setY(float) : void
+addX(float) : void
+addY(float) : void
Velocity
-dx : float
-dy : float
+getDx() : float
+getDy() : float
+setDx(float) : void
+setDy(float) : void

Core Requirements

  1. Overload the following operators as non-member functions. Then, un-comment the section of code in ta12.cpp marked Part I.

    • Velocity + Velocity : Add dx to dx and dy to dy.
    • Velocity += Velocity : Add the dx and dy of the value on the right to the current object.
  2. Overload the following operators as non-member friend functions. Then, un-comment the section of code in ta12.cpp marked Part II.

    • Velocity - Velocity : Subtract dx from dx and dy from dy.
    • Velocity -= Velocity : Subtract the dx and dy of the value on the right from the current object.
  3. Overload the following operators as member functions of the Point class. Then, un-comment the section of code in ta12.cpp marked Part III.

    • Point += Velocity : Add the dx and dy values of the Velocity object to the x and y values of the Point.

When you complete these requirements, show your code to your instructor to "pass it off."

Stretch Challenges

After completing the above steps. Make sure that everyone on the entire team is to this point and understands the material. Then, if you have time, move onto the following stretch challenges.

  1. Overload the following operators. Then, un-comment the section of code marked, Stretch I.
    • Velocity == Velocity : Returns true if the dx and dy values on the left are within 0.001 of the values on the right. (Thought question: Why not check if they are exactly the same?)
    • Velocity != Velocity : Opposite of the == operator.
    • Velocity > Velocity : Returns true if the magnitude of the velocity on the left is greater than the one on the right.
    • Velocity < Velocity : Similar to above.
    • Velocity >= Velocity : Similar to above.
    • Velocity <= Velocity : Similar to above.

    Don't forget that once you have implemented an operator, you can use it to implement the others!

  2. Overload the following operators. Then, un-comment the section of code marked, Stretch II.
    • cout << Velocity : Output the Velocity in the format: (dx=1.23, dy=4.56)
    • cin >> Velocity : Reads in a velocity in the format: 1.23 4.56
  3. Overload the following operators. Then, un-comment the section of code marked, Stretch III.
    • ++Velocity : Adds one to both the dx and dy and returns a reference to the object.
    • Velocity++ : Adds one to both the dx and dy and returns a copy of the old object.
