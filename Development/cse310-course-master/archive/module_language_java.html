Language Module—Java

Overview

The Java language is an object-oriented language that is used for embedded and application development. Java requires the use of a compiler and a Java Virtual Machine (JVM) which can both be obtained by installing the Java Development Kit (JDK). Compiled code (often with a .class or .jar extension) will run on any computer that has a JVM. Java contains a substantial library which includes many classes for data structures (Java Collection Framework), file management, and threading. In Java, every file contains a single class. A static main function is created in one of the classes and is used to provide a starting point for the program.

You can read more on sites such as the following:

  • Java—Wikipedia (open collaboration site)

  • Java Tutorial—W3Schools (tutorial site)

  • Java Reference—Oracle (official site)

  • Java Collection Frameworks—BeginnersBook (tutorial site)

Module Requirements

Implementation

You need to complete the following:

  • Write three separate programs that demonstrate the following:

    • Variables

    • Expressions

    • Conditionals

    • Loops

    • Functions

    • Classes

    • Data structure from Java Collection Framework (such as ArrayList, TreeSet, and HashMap)

  • Each program does not need to demonstrate all the items above. Collectively, however, the three programs should cover all the programming topics.

  • Each program should perform a useful task. The programs should be interesting and complex enough so that you will be excited to share them with potential employers and co-workers. In general, each program should be at least fifty lines of code.

  • You can rewrite programs you originally wrote in a different programming language (including a previous language module for this course) to complete this module.

Documentation

The programs need to be published to three separate (not one) public GitHub repositories as part of your personal software portfolio. Be careful not to include any passwords or keys in your published material. Ensure that the repository has a good description and is tagged appropriately. The README.md file in each repository should be well-documented, with at least the following information:

  • A description of your program.

  • How to set up the development environment, including all tools. Provide detailed steps so that you or another engineer could replicate the environment.

  • How to build, execute, and use your program as applicable for your development environment.

  • Screenshots of your software running.

  • A list of useful websites that you found to assist your development work.

Stretch Challenges

After you complete the module requirements, you can earn additional bonus points by completing one of the following stretch challenges:

  • Create or modify a program that reads and writes files.

  • Create or modify a program that demonstrates inheritance using the extends and abstract keywords.

  • Create or modify a program that uses an interface using the implements keyword.

The implementation of a stretch challenge must be done within the same GitHub repository, and reflected in the descriptions found in your README.md file.

You may not be awarded bonus points if the module requirements were not completed satisfactorily.

Guidance

To set up your development environment for Java, complete the following:

  • Install the latest version of either JDK (commercial) or OpenJDK (open source) from Oracle. In industry, there are license considerations between these two options. For your work in this course, either choice is acceptable. Make sure that the Java compiler and virtual machine tools are in your environment path.

  • You can either integrate Java into Visual Studio Code or run the Java tools (javac and java) from within the Visual Studio Code terminal window. As another option, you can use a tool like IntelliJ (free community version) to develop and run Java code.

There are other development environments that are provided completely online (e.g. repl.it), but this module requires that you install the required development environment on your computer. Online environments can be useful for learning. However, demonstrating the ability to setup your environment will better prepare you for the industry.
