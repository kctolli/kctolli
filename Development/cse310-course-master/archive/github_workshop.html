GitHub Workshop

Overview

During class this week we will conduct a GitHub Workshop. This workshop covers some of the content of the W01-Prove. The video in W01-Prepare covers this same material. Since all modules in this course require the publishing of public GitHub repository, the techniques in this workshop must be well understood.

During the workshop, we will be writing a simple Hello World program in Python using Visual Studio Code. We will then publish the code to a GitHub repository.

  1. Install Python 3.x

  2. Install Visual Studio Code

  3. Install Git

  4. Create GitHub account

  5. Create folder for Hello World Example

  6. Open the Hello World Example folder in Visual Studio Code

  7. Create a Python file and let Visual Studio Code auto-detect Python

  8. Run Hello World

  9. Publish Hello World to GitHub and view the published code within GitHub

  10. Change Code, Commit Changes, Publish Changes to GitHub

  11. Create README.md using Markdown

  12. Publish documentation, view in GitHub, add tag in GitHub

The demonstration is also available by watching the GitHub Workshop video.

Tool Preparation

Since this workshop will be done during the first week of the semester, there is no preparation required. You will want to review the previously mentioned video as you implement the first assignment.
