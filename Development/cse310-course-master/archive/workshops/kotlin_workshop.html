Kotlin Workshop

Overview

During class this week we will conduct a Kotlin Workshop. This workshop covers some of the content of the Kotlin Language Module. If you want to learn more about Kotlin, then you encouraged to select and implement this module during this semester.

We will complete the first two examples using REPL. The third example will use the IntelliJ IDE and a GitHub repository.
