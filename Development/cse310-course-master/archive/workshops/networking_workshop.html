Networking Workshop

Overview

During class this week we will conduct a Networking Workshop using Python. This workshop covers some of the content of the Networking Discovery Module. If you want to learn more about Networking, then you encouraged to select and implement this module during this semester.

We will complete the examples using Visual Studio Code and GitHub repositories.
