CSE 310 | Applied Programming

Android App Workshop

Overview

During class this week we will conduct an Android App Workshop using Kotlin. This workshop covers some of the content of the Android App Discovery Module. If you want to learn more about Android, then you encouraged to select and implement this module during this semester.

We will complete the examples using Android Studio and GitHub repositories. The first example does not have starting code.
