Cloud Database Workshop

Overview

During class this week we will conduct a Cloud Databse Workshop. This workshop covers some of the content of the Cloud Database Discovery Module. If you want to learn more about cloud databases, then you encouraged to select and implement this module during this semester.

During the workshop, we will be using the Firestore database which will use external libraries from Google. All of the examples will include the use of a private key for the database. Traditionally, these keys should not be shared to avoid other people writing software to manipulate your database. We will complete the following examples using REPL:
