W14 Teach: Team Project

Team Meeting

You do not need a team meeting this week. You should coordinate together to finish the project as needed. There is no final group delivery. The W14 Prove is an individual report that each team member will complete confidentially. This final report will receive the grade for the project as described in the Team Project Description rubric. It would be helpful to remind your team members to complete their individual report before the end of the semester.

You are also encouraged to make a copy (if the repository was not created in your account) of the project by doing a fork in GitHub (GitHub Forking).
